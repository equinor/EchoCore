Class AuthenticationProvider

Echo Core Authentication provider class for creating providers that provide logout-, login- and getAccessToken-methods. Can be extended if needed Based on the @azure/msal-browser package

param

@azure/msal-browser Configuration object used for authentication

param

Silent request used for login, can be specified if default is not not enough:

var

defaultLoginRequest = { account: user, forceRefresh: false, scopes: ['openid', 'profile', 'User.Read', 'offline_access'] };
